# 🎯 قواعد التصميم الأساسية

## المبادئ الثابتة

```
✅ RTL First      - التصميم من اليمين لليسار أولاً
✅ Mobile First   - التصميم للموبايل ثم التوسع
✅ Consistency    - اتساق في كل الشاشات
✅ Accessibility  - سهولة الوصول والاستخدام
✅ Performance    - سرعة التحميل والاستجابة
```

## قاعدة 8px Grid

جميع المسافات والأحجام يجب أن تكون من مضاعفات 8:

```
8px  = 0.5rem = spacing-2
16px = 1rem   = spacing-4
24px = 1.5rem = spacing-6
32px = 2rem   = spacing-8
40px = 2.5rem = spacing-10
48px = 3rem   = spacing-12
```

# 📝 نظام Typography

## الخطوط

```css
/* الخط الأساسي */
font-family: "Cairo", sans-serif;

/* للأرقام الإنجليزية */
font-family: "Inter", sans-serif;
```

## أحجام الخطوط

| الاسم | الحجم | الاستخدام             | Tailwind    |
| ----- | ----- | --------------------- | ----------- |
| xs    | 12px  | تفاصيل صغيرة، badges  | `text-xs`   |
| sm    | 14px  | نص ثانوي، hints       | `text-sm`   |
| base  | 16px  | النص العادي           | `text-base` |
| lg    | 18px  | عناوين فرعية          | `text-lg`   |
| xl    | 20px  | عناوين البطاقات       | `text-xl`   |
| 2xl   | 24px  | عناوين الصفحات        | `text-2xl`  |
| 3xl   | 30px  | عناوين رئيسية         | `text-3xl`  |
| 4xl   | 36px  | أرقام كبيرة (الأسعار) | `text-4xl`  |

## وزن الخط (Font Weight)

| الوزن | الاستخدام     | Tailwind         |
| ----- | ------------- | ---------------- |
| 400   | النص العادي   | `font-normal`    |
| 500   | نص مهم        | `font-medium`    |
| 600   | عناوين        | `font-semibold`  |
| 700   | عناوين رئيسية | `font-bold`      |
| 800   | تأكيد قوي     | `font-extrabold` |

## Line Height

```css
leading-tight: 1.25; /* للعناوين */
leading-normal: 1.5; /* للنصوص */
leading-relaxed: 1.625; /* للفقرات الطويلة */
```

# 🧱 المكونات الأساسية (Components)

## 1. الأزرار (Buttons)

### أنواع الأزرار

```
Primary   → العمليات الرئيسية (إضافة، حفظ)
Secondary → العمليات الثانوية (إلغاء، رجوع)
Success   → تأكيد/إكمال (الدفع، تأكيد)
Danger    → حذف/إلغاء خطير
Outline   → أزرار ثانوية بحدود
Ghost     → أزرار شفافة للأيقونات
```

### الأحجام

```
sm  → داخل الجداول والقوائم
md  → الاستخدام العادي (افتراضي)
lg  → الأزرار المهمة
xl  → زر الدفع الرئيسي
```

### الكود

```tsx
// Button Component API
<Button
  variant="primary|secondary|success|danger|outline|ghost"
  size="sm|md|lg|xl"
  isLoading={boolean}
  disabled={boolean}
  leftIcon={ReactNode}
  rightIcon={ReactNode}
>
  النص
</Button>
```

## 2. حقول الإدخال (Inputs)

### الهيكل الثابت

```
┌─────────────────────────────┐
│ Label                       │
├─────────────────────────────┤
│ ┌─────────────────────────┐ │
│ │ Input field             │ │
│ └─────────────────────────┘ │
│ Hint text / Error message   │
└─────────────────────────────┘
```

### الكود

```tsx
<Input
  label="اسم المنتج"
  type="text|email|password|number"
  placeholder="أدخل اسم المنتج"
  error="رسالة الخطأ"
  hint="نص مساعد"
  required
/>
```

## 3. البطاقات (Cards)

### أنواع البطاقات

```
card       → بطاقة عادية
card-hover → بطاقة تفاعلية (للمنتجات)
```

### الهيكل

```
┌─────────────────────────────┐
│ ┌─────────────────────────┐ │
│ │       Image/Icon        │ │ ← اختياري
│ └─────────────────────────┘ │
│                             │
│ Title                       │
│ Description                 │ ← اختياري
│                             │
│ ┌─────────────────────────┐ │
│ │ Actions / Price         │ │ ← اختياري
│ └─────────────────────────┘ │
└─────────────────────────────┘
```

## 4. النوافذ المنبثقة (Modals)

### الأحجام

```
sm   → 384px  → تأكيدات بسيطة
md   → 448px  → فورم صغير
lg   → 512px  → فورم متوسط
xl   → 576px  → فورم كبير
full → 896px  → محتوى معقد
```

### الهيكل

```
┌───────────────────────────────────┐
│ Header (Title + Close)            │
├───────────────────────────────────┤
│                                   │
│           Content                 │
│                                   │
├───────────────────────────────────┤
│ Footer (Actions)                  │ ← اختياري
└───────────────────────────────────┘
```

## 5. الجداول (Tables)

### الهيكل

```tsx
<div className="table-container">
  <table className="table">
    <thead>
      <tr>
        <th>العمود</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td>القيمة</td>
      </tr>
    </tbody>
  </table>

```

## 6. Badges / Tags

### الأنواع

```
badge-primary  → معلومات عامة
badge-success  → حالة ناجحة
badge-warning  → تحذير
badge-danger   → خطأ/ملغي
badge-gray     → محايد
```

# 📜 قواعد الكود

## 1. تسمية الملفات

```
components/    → PascalCase.tsx      (Button.tsx, ProductCard.tsx)
pages/         → PascalCase.tsx      (LoginPage.tsx, POSPage.tsx)
hooks/         → camelCase.ts        (useAuth.ts, useCart.ts)
utils/         → camelCase.ts        (formatters.ts, constants.ts)
types/         → kebab-case.types.ts (auth.types.ts, product.types.ts)
api/           → camelCase.ts        (authApi.ts, productsApi.ts)
slices/        → camelCase.ts        (authSlice.ts, cartSlice.ts)
```

## 2. هيكل المكون

```tsx
// 1. Imports
import { useState } from "react";
import clsx from "clsx";

// 2. Types
interface ComponentProps {
  prop1: string;
  prop2?: number;
}

// 3. Component
export const Component = ({ prop1, prop2 = 0 }: ComponentProps) => {
  // 3.1 Hooks
  const [state, setState] = useState(false);

  // 3.2 Handlers
  const handleClick = () => {
    // ...
  };

  // 3.3 Render
  return <div className="...">{/* Content */}</div>;
};
```

## 3. قواعد Tailwind

```tsx
// ✅ Good - استخدم clsx للـ conditional classes
<div className={clsx(
  'base-classes',
  condition && 'conditional-class',
  {
    'class-a': conditionA,
    'class-b': conditionB,
  }
)} />

// ❌ Bad - template literals
<div className={`base ${condition ? 'a' : 'b'}`} />
```

## 4. قواعد RTK Query

```tsx
// ✅ استخدم الـ hooks المُولدة تلقائياً
const { data, isLoading, error } = useGetProductsQuery();
const [createProduct, { isLoading: isCreating }] = useCreateProductMutation();

// ✅ استخدم skipToken للـ conditional queries
const { data } = useGetProductQuery(productId ?? skipToken);
```

## 5. قواعد Redux

```tsx
// ✅ استخدم typed hooks
import { useAppDispatch, useAppSelector } from "@/store/hooks";

const dispatch = useAppDispatch();
const user = useAppSelector(selectCurrentUser);

// ✅ استخدم selectors
export const selectTotal = (state: RootState) =>
  state.cart.items.reduce(
    (sum, item) => sum + item.quantity * item.product.price,
    0
  );
```

## 6. قواعد Error Handling

```tsx
// ✅ Always handle errors
try {
  await mutation(data).unwrap();
  toast.success("تم بنجاح");
} catch (error) {
  toast.error("حدث خطأ");
}

// ✅ Show loading states
{
  isLoading && <Loading />;
}
{
  error && <ErrorMessage message={error.message} />;
}
{
  data && <Content data={data} />;
}
```
